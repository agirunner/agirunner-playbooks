# Policy Review

## Use When

Use this playbook when a team needs a policy document turned into an
operationally useful review instead of a vague summary.

## Inputs

- policy subject
- review goal
- policy text or source material context

## Output

The workflow should produce:
- scope and applicability
- obligations and prohibitions
- ambiguities or conflicts
- process impact
- recommended follow-up

## Workspace Notes

This playbook is storage-agnostic and does not require a writable
workspace to complete honestly.

## Import Notes

This playbook is for operational and compliance review. It should not be
treated as a substitute for legal advice.
