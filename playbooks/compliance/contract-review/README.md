# Contract Review

## Use When

Use this playbook when a team needs a risk-focused readout of a contract
or agreement before negotiation, approval, or implementation planning.

## Inputs

- contract subject
- review goal
- counterpart or deal context if available

## Output

The workflow should produce:
- key obligations
- risk areas
- dates and deadlines
- open questions
- recommended follow-up

## Workspace Notes

This playbook is storage-agnostic and can complete from document
artifacts alone.

## Import Notes

This playbook helps with structured review and follow-up. It should not
be treated as a substitute for legal counsel.
