# Content Pipeline

## Use When

Use this playbook when a content brief needs to become a reviewed draft
that is ready for publication, editorial handoff, or further production.

## Inputs

- content goal
- target audience
- source material context

## Output

The workflow should produce:
- framed brief
- drafted content
- editorial review findings
- important limits or caveats
- next publication or handoff step

## Workspace Notes

This playbook works with a writable content workspace or in advisory
mode.

## Import Notes

This playbook benefits from source material that is already organized,
but it should still close honestly with explicit gaps when the brief is
incomplete.
